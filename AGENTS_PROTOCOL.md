# Antigravity 7-Agent System Protocol

## Overall Mission

Build, validate, optimize, and refine a full-stack project using a coordinated 7‑agent system across multiple workspaces.

## The 7 Agents

### 1. Architecture & Strategy Agent (Agent‑0: Commander)

**Role:** Lead planner that sets the foundation for ALL other agents.
**Responsibilities:**

- Create the system blueprint
- Decide frameworks, libraries, design patterns
- Generate file structure
- Define environment variables, config, dev/prod modes
- Coordinate workflow order
- Validate downstream decisions

### 2. Backend API Agent (Agent‑1)

**Role:** Build and maintain backend logic.
**Responsibilities:**

- Build API endpoints, routing, validation
- Implement business logic
- Integrate authentication + middleware
- Create database models, migrations
- Provide API spec

### 3. Frontend/UI Agent (Agent‑2)

**Role:** Build user interface and integrate the backend.
**Responsibilities:**

- Build UI components, routing, layout
- Connect to backend API
- Manage global state and user session
- Render dashboards, forms, visualizations

### 4. Database & DevOps Agent (Agent‑3)

**Role:** Ensure stability across environments.
**Responsibilities:**

- Set up database
- Manage migrations
- Docker/Environment config
- CI/CD pipelines

### 5. Testing & QA Agent (Agent‑4)

**Role:** Validate everything.
**Responsibilities:**

- Unit, Integration, E2E tests
- Bug verification

### 6. Refactor & Optimization Agent (Agent‑5)

**Role:** Clean up and optimize.
**Responsibilities:**

- Code cleanup
- Performance improvements
- Security strengthening
- Standardization

### 7. UX & Documentation Agent (Agent‑6)

**Role:** Finalize product.
**Responsibilities:**

- Improve UX flow
- Generate documentation (README, API docs)
- Refine UI look & feel

## Execution Flow

1. **Phase 1 - System Setup (Agent-0):** Blueprint, structure, scaffolding.
2. **Phase 2 - Core Build (Agents 1, 2, 3):** Parallel execution of Backend, Frontend, DevOps.
3. **Phase 3 - Verification (Agent-4):** Testing.
4. **Phase 4 - Optimization (Agent-5):** Refactoring.
5. **Phase 5 - Finalization (Agent-6):** Docs and UX.
